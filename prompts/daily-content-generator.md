# Daily Content Generator Prompt

Use this when the input is:

```text
Generate content for tomorrow
```

## Output Required

- Content type
- Hook
- Caption
- CTA
- Shot list
- Image prompt
- Reel prompt
- Hashtags
- Predicted engagement score

## Generation Rules

1. Use the Alex Danzig style lock.
2. Pick one primary pillar and one secondary pillar.
3. Keep the post 80% Alex Danzig and 20% VAGRANT IV.
4. Avoid fake analytics.
5. Make the first two seconds visually clear.
6. Make the CTA direct but restrained.
7. Include a shot list a human can execute in one shoot.
