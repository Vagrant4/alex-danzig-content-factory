# Viral Reel Analyzer Prompt

Use this to convert manually observed viral reels into Alex Danzig content ideas.

## Input

- Creator handle
- Reel URL
- Hook
- Reel structure
- CTA
- Actual performance notes, if available

## Output

- Reusable hook pattern
- Shot sequence pattern
- Caption structure
- CTA structure
- Alex Danzig version
- VAGRANT IV product integration angle

## Rules

- Do not claim content is viral unless real performance data is entered.
- If performance data is missing, label the observation as manually selected.
- Never copy a creator's exact creative expression.
- Extract structure, then create an original Alex/VAGRANT IV version.
